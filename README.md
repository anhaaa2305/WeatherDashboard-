# WeatherDashboard
# Install
Step 1: Clone project by URL: git clone  https://github.com/anhaaa2305/WeatherDashboard-.git
Step 2: Open the project with Android Studio.
Step 3: Config SDK (Flutter & Android). I am using FlutterSDK version: 3.22.2 & Android SDK using the recommend.
Step 4: Run "flutter doctor" and make sure no issues are found.
Step 5: Connect with a physical device or emulator, if you use Chrome, choose Destination is Chrome in the Top bar.
Step 6: Open the terminal at the root folder of the project, run "flutter run," and choose the destination Device.

# If the project doesn't work with the current APIKey, change it to constant.dart and generate a new APIKey from https://www.weatherapi.com.
